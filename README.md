## Overview
Next.js + Tailwind frontend for ProjectPulse:
- Role-based dashboards (Admin / Employee / Client)
- Weekly check-ins, client feedback
- Project health tracking & activity timeline

## Tech Stack
- Next.js, Tailwind CSS
- Axios, Framer Motion
- Recharts / Chart.js (optional)

## Features
- Login with JWT, role-based access
- Admin: manage projects & risks
- Employee: submit weekly check-ins
- Client: submit feedback & flag issues
- Responsive UI, loading & error states

## Getting Started
```bash
git clone <frontend-repo-url>
cd frontend
npm install
npm run dev
```
